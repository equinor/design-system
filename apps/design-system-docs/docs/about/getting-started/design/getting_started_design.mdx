# Getting started with design

Set up Figma, connect to the EDS component libraries, and start designing with shared tokens and components.

**Prerequisites:** Equinor Figma account

</div>
<div className="about-section about-section--surface">

## Set up Figma

1. **Get Figma access** — apply through AccessIT or contact your team administrator
2. **Join your design team** — search for your team in Figma and request access, or ask your team admin to add you
3. **Enable the EDS library** — open your file's Assets panel and enable the EDS component library

For detailed permissions guidance and workspace setup, see the [Figma guide](figma.md).

</div>
<div className="about-section">

## Configure your workspace

Before you start designing, adjust a few Figma settings:

- Change nudge amount to **8px** in Figma preferences (matches the EDS spacing grid)
- Create frames using standard screen sizes — **1440px** width for desktop
- Familiarise yourself with the EDS component naming in the Assets panel

</div>
<div className="about-section about-section--surface">

## Build your first interface

Start simple and layer in complexity:

1. Add a layout structure — use `TopBar` or `SideSheet` for navigation
2. Add content — use `Typography` components for headings and body text
3. Add interactive elements — `Button`, `TextField`, or `Checkbox`
4. Add icons from the EDS icon library where needed

### Accessibility checklist

- Use EDS colour combinations to meet **4.5:1** contrast for normal text and **3:1** for large text (WCAG 2.1 AA)
- Set focus states for all interactive elements
- Ensure touch targets are at minimum **44px**

</div>
