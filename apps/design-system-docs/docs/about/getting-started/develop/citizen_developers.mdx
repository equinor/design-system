## Power Apps

A canvas app template with pre-configured components, layout, theming, and interaction patterns aligned to EDS guidelines.

What's included:

- EDS colour, spacing, and typography tokens where the platform supports them
- Common screen patterns — dashboard, form, list, detail
- Accessible defaults — labels, focus order, contrast guidance
- Clearly named components for reuse and replacement
- Documentation of customisation boundaries and extension steps

Import the template, review the built-in guidance, then tailor the documented configuration areas.

**Preview:** [Canvas Apps Template](https://apps.equinor.com/edspowerapps) (Equinor network)

## Power BI

Light-themed report templates with pre-configured charts, layouts, and styling that follow EDS principles.

### How to use

1. Download the template (`.pbix`) or theme (`.json`) below
2. Open the `.pbix` file in Power BI Desktop, or apply the `.json` theme via **View → Themes → Browse for themes**
3. Replace the sample data with your own data source
4. Adjust chart titles and labels — the template colours and fonts will carry over

**Preview:** [Power BI Preview](https://apps.equinor.com/edspbi) (Equinor network)

**Template:** [Power BI EDS Template (PBIX)](pathname:///assets/eds-pbi-template-v2.pbix)

**Theme:** [Power BI EDS Theme (JSON)](pathname:///assets/eds-pbi-theme-v2.json)

## Working with EDS templates

These guidelines are specific to customising the Power Platform templates:

- **Start from the template** — don't build from scratch. The templates handle colour, spacing, and font configuration.
- **Stay within documented boundaries** — each template documents which areas are safe to customise and which should not be changed.
- **Use the EDS theme JSON** for any new Power BI reports — it applies the correct colour palette and font stack automatically.
- **Test on different screen sizes** — Power Apps canvas apps need responsive adjustments for tablet vs. desktop.
- **Check colour contrast** if you add custom colours — use the built-in EDS palette where possible, as it already meets WCAG 2.1 AA.
