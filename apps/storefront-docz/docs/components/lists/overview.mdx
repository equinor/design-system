_Lists_ organise information into related groups of content.

<!--One to two sentences as an introduction to the component-->

## Guidelines

<!--List how to use it, sizes, widths, grid interactions, color allowances -->

### Bullet lists

Bullet lists are used when order is not of importance.

### Numbered lists

Numbered lists are used when order and hierarchy is needed.

<FigmaImage
  url="https://www.figma.com/file/0bGXR2sCwMVSDNyyzu5BXrO5/UI%E2%80%94User-Interface?node-id=1%3A10"
  alt="Figma design sketches of lists"
/>

**Don't**

<!--Bullet point list if needed -->

- Do not use lists to organise large amounts of information, use a _Table_ instead.
- Do not use lists as navigation.
