# Using Companion by Tokens Studio [​](#using-companion-by-tokens-studio)

Companion by Tokens Studio is a Figma plugin which is meant purely for consumption of design tokens from Studio as variables in Figma.

**Prerequisites**:

-   [Api Keys](./../settings/api-keys)

### Step 1: Launch the Companion Plugin in Figma [​](#step-1-launch-the-companion-plugin-in-figma)

![](/images/Screenshot%202025-11-21%20at%207.11.21%E2%80%AFPM.png)

-   Open a Figma design file (a new or existing one is fine).​
-   Go to the Figma “Plugins” menu and launch **Companion by Tokens Studio**.

### Step 2: Install/Launch the plugin [​](#step-2-installlaunch-the-plugin)

1.  Go to plugins (ctrl+k) > Companion by Tokens Studio.
2.  Run the plugin.

![](/images/Screenshot%202025-11-21%20at%207.13.03%E2%80%AFPM.png)

### Step 3: Setting up Studio sync [​](#step-3-setting-up-studio-sync)

1.  Return to Studio.
2.  Click on Find your API key. You can also jump to the API keys page by using the keyboard shortcut cmd+k.
3.  The API key is linked to the user which means that it gives access to all the Organisations and Projects that a user is part of.

For more info read [Platform > API keys](./../settings/api-keys).

### Step 4: Syncing to Companion by Tokens Studio [​](#step-4-syncing-to-companion-by-tokens-studio)

1.  Return to Figma and the plugin.
2.  Enter your API key.
3.  Choose the Organisation that you want to connect.
4.  Choose the Project that you want to connect.
5.  You are now connected to Studio and your tokens should reflect in the plugin.

![](/images/Api%20key%20to%20companion.gif)

For more information on the features of Companion by Tokens Studio, see [Companion by Tokens Studio](./../plugins/companion-by-tokens-studio).
