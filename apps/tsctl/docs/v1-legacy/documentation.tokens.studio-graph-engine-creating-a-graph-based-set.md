# Creating a Graph Based Set [​](#creating-a-graph-based-set)

In Studio, you can create different types of sets to manage your design tokens effectively. This guide walks you through the process of creating a **graph-based set**.

### Step 1: **Navigate to the Tokens Module** [​](#step-1-navigate-to-the-tokens-module)

-   Open Studio.
-   In the **left panel**, click on the **Tokens module**.

![](/images/CleanShot%202025-02-28%20at%2019.53.05@2x.png)

### Step 2: **Access the Sets Panel** [​](#step-2-access-the-sets-panel)

-   Here, you will see the existing token sets in your project.
-   Click on **Create New Set**.

### Step 3: **Choose the Set Type** [​](#step-3-choose-the-set-type)

-   A dialog will appear, prompting you to choose between:
    -   [**Static Set**](./../../platform/tokens/token-sets#static-sets)(manually managed tokens).
    -   [**Graph-Based Set**](./../../platform/tokens/token-sets#graph-based-sets) (dynamic token relationships using graph logic).
-   Select **Graph-Based Sets.**

### Step 4: **Configure Your Graph-Based Set** [​](#step-4-configure-your-graph-based-set)

-   **Enter a Name** for the set.
-   _(Optional)_ Add a **Description** to clarify its purpose.

### Step 5: **Create the Set** [​](#step-5-create-the-set)

-   Click **Create**, and your graph-based set will be generated.

![](/images/CleanShot%202025-02-28%20at%2020.07.05.gif)

### Step 6: **Next Steps** [​](#step-6-next-steps)

-   The newly created graph-based set is now available in the UI.
-   You can now start defining relationships, nodes, and logic within the graph editor.
