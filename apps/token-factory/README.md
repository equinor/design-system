# Token Factory

Workshop-companion game that teaches how `@equinor/eds-tokens` is built and consumed. Designed for one driver to play in front of the team while everyone narrates.

See [PLAN.md](./PLAN.md) for art direction, phases, and progress.

## Run locally

```sh
pnpm install
pnpm --filter @equinor/token-factory dev
```

Opens at <http://localhost:5180>.

## Status

Phase 0 — scaffolding. Empty factory stage with one placeholder floor tile and one real EDS token swatch as proof the pipeline works end-to-end.
