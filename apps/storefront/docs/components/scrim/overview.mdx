import { Scrim, Button, Typography } from '@equinor/eds-core-react'

A scrim is a temporary visual effect that fades the general interface while allowing the user to focus on an overlay.

## Guidelines

Always use a scrim with an overlay component such as a dialog, elevated sidesheet or elevated navigation drawer (mobile only).

  url="https://www.figma.com/file/0bGXR2sCwMVSDNyyzu5BXrO5/Desktop-UI?node-id=85%3A32"

## Interactions

When the scrim is clicked on or the `Esc` key is pressed---the overlay component will close, closing the scrim with it.

## Demo

<Playground>
  {() => {
    const [visibleScrim, setVisibleScrim] = React.useState(false)
    const handleClose = (event, closed) => {
      if (closed) {
        setVisibleScrim(closed)
      } else {
        setVisibleScrim(!visibleScrim)
      }
    }
    return (
      <>
        <div>
          <Button onClick={() => setVisibleScrim(true)}>Trigger Scrim</Button>
        </div>
        {visibleScrim && (
          <Scrim onClose={handleClose} isDismissable>
            <div style={{ backgroundColor: '#fff', padding: '1rem' }}>
              <Typography variant="body_short">
                Press close or hit 'ESC' to close scrim.
              </Typography>
              <Button onClick={() => setVisibleScrim(false)}>Close</Button>
            </div>
          </Scrim>
        )}
      </>
    )
  }}
</Playground>
