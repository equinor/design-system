import { SingleSelect, MultiSelect, NativeSelect } from '@equinor/eds-core-react'

The _Select_ component allows users to choose one or multiple items or options from a list.


## Guidelines

A select can be used in different settings: 
* to filter or sort content on a page
* to submit data


## Singleselect versus multiselect variant

The `singleselect` variant is used to choose only one option from a list.
The `multiselect` variant is used if multiple choices can be applied.

A native select component is also available for implementation. We recommend using native select on product with limited space as it provide a better user experience.

### Best practices

Use clear and simple language in the placeholder text. 
It will help the user to understand what they are expected to choose from.

Order your list of options in a way that will make the most sense to the user: 
* alphabetically
* numerically
* by most commonly selected option

Display a maximum of six items/options at a time.

### Implementation

A native select component is also available for implementation. We recommend using native select on products with limited space as it provides a better user experience.

## Demo

### SingleSelect

<Playground>
{() => {
  const items = [
  'Oslo',
  'Rogaland',
  'Møre og Romsdal',
  'Nordland',
  'Viken',
  'Innlandet',
  'Vestfold og Telemark',
  'Agder',
  'Vestland',
  'Trøndelag',
  'Troms og Finnmark',
]
  return (
    <div
        style={{
          display: 'grid',
          gridGap: '32px',
         
        }}
      >
      <SingleSelect label="Where do you live?" items={items} />
      <SingleSelect
        label="I have a preselected option"
        initialSelectedItem="Troms og Finnmark"
        items={items}
      />
      <SingleSelect label="Choose an element" meta="km/t" items={items} disabled />
    </div>
  )
}}
</Playground>
