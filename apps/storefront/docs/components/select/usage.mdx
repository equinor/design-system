## Figma

### How to add

1. Locate the **Assets** tab in the **Layers Panel**.
2. Use the search bar to search for select.
3. Once the component needed is located, click and drag it into the frame/artboard.

### How to use

1. Place the selected component on the frame/artboard and choose a _variant_ in the **Inspector Panel**.
   - By default, the component is set to Light theme/Single select/Enabled.
2. From the **Inspector Panel**, choose the variants of the component:
   - Light or Dark theme
   - Single or Multiselect
   - State (Enabled/Focus/Hover/Active/Selected/Disabled)
