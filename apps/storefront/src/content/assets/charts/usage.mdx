## Figma


### How to add

1. Right click anywhere on the page you need a chart. 
2. Navigate the menu that opens to Plugins > Charts. Example: https://i.imgur.com/uAYKlM1.png
3. In the chart popup, choose the type of chart needed. 
3. Configure the chart. 
4. Once the chart looks correct, click "Add chart" at the bottom of the popup. 
5. This will place the chart on centere of the page. Move the chart to the frame/artboard it is needed on. 


### How to use

1. Select the Chart in the frame/artboard. 
2. Locate and expand the highlighted *layer* in the **Layers Panel**.
3. Select all the text layers, apply the text style `UI/Chart` to all of the layers. 
4. Select all the grid layers, apply the colour style `UI background/Medium` to all of the layers. 
5. Select each line or bar, and apply a different color style from `Infographic Primary` or `Infographic substitute` to each line or bar.
