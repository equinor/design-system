## Figma

### How to add

1. Locate the **Assets** tab in the **Layers**.
2. There are two ways to locate the `Logo` components needed. 
	* Using the search bar, searching for a component grouping name or variation name.
	* Scrolling through the folders list, opening the grouping needed.
3. Once the component needed is located, click and drag it into the frame/artboard. 


### How to use


1. Select the `Logo` in the frame/artboard. 
2. When resizing the `Logo` always hold the `Shift` key to keep the correct aspect ratio.
