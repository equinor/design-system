## Figma

### How to add

1. Locate the [Assets tab :warning:]() in the *Layers Panel*.
2. There are alternatives to locate the component needed. 
	* Using the search bar, searching for a component grouping name or variation name.
	* Scrolling through the folders list, opening the grouping needed.
3. Once the component needed is located, click and drag it into the frame/artboard. 


### How to use

1. Select the `Image placeholder` in the frame/artboard. 
2. Locate the [Design tab :warning:]() in the *Inspector Panel*.
3. Under the *Fill* section, click on the *Image* to expand the image options menu. 
4. Hover over the current image and click *Choose image...* to select a new image. 
5. When resizing `Image placeholder` always holds the `Shift` key to keep the correct aspect ratio.
