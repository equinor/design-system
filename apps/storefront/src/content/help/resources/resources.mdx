## Figma

To get access to the EDS in Figma, you need to apply in [Access IT](//accessit.equinor.com/Search/Search?term=figma).

## Github

The EDS repository will be public at some time – but at the moment you need to apply for access to Github in [Access IT](//accessit.equinor.com/Search/Search?term=github).

## NPM

The [EDS modules in NPM](https://www.npmjs.com/search?q=%40equinor%2Feds-) are public and available for anyone to install, see the individual README’s for further details.

## CDN

See [getting started for developers](/getting-started/developers/) for how to use the CDN.

## Slack

The official EDS channel in the Equinor Slack is [#eds-design-system](//app.slack.com/client/T02JL00JU/CJT20H1B9). Please reach out to us if you want to discuss something or if you just want to say hi.
