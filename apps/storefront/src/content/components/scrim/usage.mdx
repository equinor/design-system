Figma

How to add
Select "Assets" (link to glossary) and find the scrim element by either scrolling to the "Scrim" section or search for "scrim".  Add the scrim by dragging it to your project.

How to use
Resize the scrim to the size you need, provided you follow grid guidelines. Scrim usage is described in the "Scrim" page under "Components".
