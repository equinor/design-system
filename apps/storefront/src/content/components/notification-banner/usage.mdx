## Figma

### How to add

1. Locate the **Assets** tab in the **Layers Panel**.
2. There are two ways to locate the `Notification: Banner` components needed. 
	* Using the search bar, searching for a component grouping name or variation name.
	* Scrolling through the folders list, opening the grouping needed.
3. Once the component needed is located, click and drag it into the frame/artboard. 


### How to use


1. Place the `Notification: Banner` components needed on the frame/artboard. 
3. Locate and expand the highlighted *layer* in the **Layers Panel**.
4. Edit the text label.
5. Select all the `Notification: Banner` header layers and convert them to a frame/artboard by using the following shortcut: 
  * Mac: `Command` + `Option` + `G`
  * PC: `Ctrl` + `ALT` + `G`
6. Make sure to name the new group. 
7. Locate the **Design** tab in the **Inspector Panel**.
8. Under the **Constraints** section, set up constraints.
