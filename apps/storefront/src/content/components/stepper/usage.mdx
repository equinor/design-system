## Figma

### How to add

1. Locate the **Assets** tab in the **Layers Panel**.
2. There are two ways to locate the `Stepper` component needed.  
    * Use the search bar to search for a component grouping name or variation name.
    * Scroll through the folders list and open the grouping needed.
3. Once the component is located, click and drag it into the frame/artboard. 


### How to use

1. Place the first `Stepper` on your frame/artboard. 
2.Toggle the grid layout on/off by using the following shortcut:  
    * Mac: `Control + G`
    * PC: `Ctrl + Shift + 4
3. Locate the **Design** tab in the **Inspector Panel**.
4. Rename the title. 
5. Add a description, if needed. If not, hide the *layer*. 
6. Resize the stepper to the width and height needed. 
7. Edit the `Line` *layer*. 
    * On the `Horizontal` stepper, this is done by editing the *layer* `----`. Add or remove `-` until the line does not touch the title text by approximately 8px. 
    * On the `Vertical` stepper, this is done automatically when/if you resize.
8. If using the icon stepper, change the icon:  
    * Locate and select the `Icon > Placeholder Icon` *layer* in the **Layers Panel**.
    * Locate the **Design** tab in the **Inspector Panel**.
    * Under the **Instance** section, click the icon name to expand the menu. 
    * Choose a new icon. 
9. Repeat steps 1-8 until you have all your steps inserted. 
10. Top align the steps. Make sure to use the spacing provided in each step. 
11. Select all the stepper *layers* and convert them to a frame/artboard by using the following shortcut: 
    * Mac: `Command` + `Option` + `G`
    * PC: `Ctrl` + `ALT` + `G`
12. Make sure to name the new group. 
13. Locate the **Design** tab in the **Inspector Panel**.
14. Under the **Constraints** section, set up constraints.
