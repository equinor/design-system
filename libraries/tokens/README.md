## Installation

```sh
npm install @equinor/eds-tokens
```
## Usage

```js 
import { tokens } from '@equinor/eds-tokens'
```

## Tokens

- Clickbounds
- Colors
- Elevation
- Shape
- Spacing
- Interaction states
- Typography (`ot`, `woff` or `woff2` font required)

[design tokens]: https://css-tricks.com/what-are-design-tokens/
