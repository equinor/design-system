## Figma

### How to add
Select "Assets" (:warning: link to Getting Started > Figma, Finding components) and find system icons by either scrolling to the "System Icons" section, and opening the cateogrised folders or searching for "system icons". You can also search for the name of the icon you are looking for. Add the icon you need by dragging it to your project.


### How to use
Icons can be resized, but remeber to hold shift in order to keep proportions. Make sure to follow the guidlines set for System Icons.
