## Figma

### How to add
Select "Assets" (:warning: link to Getting Started > Figma, Finding components) and find logos by either scrolling to the Logos section or searching for logos. Add the logo you need by dragging it in.

### How to use
Logos can be resized, but remeber to hold shift in order to keep proportions. Make sure to follow the guidelines set for logos.
