## Figma

### How to add
Select "Assets" (:warning: link to Getting Started > Figma, Finding components) and find product icons by either scrolling to the "Product Icons" section or search for "product icons". Add the icon you need by dragging it in.


### How to use
Icons can be resized, but remeber to hold shift in order to keep proportions. Make sure to follow the guidelines set for product icons.
