# Tokens Studio backup — machine-generated, never merge

Hourly snapshots of every token source in Tokens Studio (project
`4952a007-699a-4124-a043-124e80cc28d6`), pulled and pushed by
`.github/workflows/tokens_studio_backup.yaml` on `main`. One directory
per source alias in `packages/eds-tokens/.studio.json`.

The platform has no undo/rollback — this branch is the recovery point
for everything between releases. Recovery instructions:
`documentation/agent-instructions/TOKENS_STUDIO.md` § Backup & recovery.

Do not edit, rebase or merge this branch.
