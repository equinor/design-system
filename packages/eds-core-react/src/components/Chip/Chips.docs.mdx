# Chip

Chips, also known as tags or badges, are advanced badges that represent discrete information.

## Usage

Chips allow users to enter information, make selections, filter content, or trigger actions.
<ul>
    <li>Chips are **compact** components that represent discrete information.</li>
    <li>Chips should be **relevant** to the content or task they represent.</li>
    <li>Chips should make tasks easier to complete, or content easier to sort.</li>
</ul>

  ```tsx
import { Chip } from '@equinor/eds-core-react'

<Chip>tag</Chip>
```
