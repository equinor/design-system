## Usage

**Selection controls**

<ul>
    <li>Allow users to select options, make decisions and set preferences.</li>
    <li>Are to be visible and understandable at a quick glance.</li>
    <li>Are always accompanied with a clear label.</li>
</ul>

**Checkbox**

<ul>
    <li>It should be visible at a glance if a checkbox has been selected, and selected items should be more visually prominent than unselected items.</li>
    <li>Checkboxes make it easy to compare available options.</li>
</ul>

```tsx
import { Checkbox } from '@equinor/eds-core-react'

<Checkbox />
```
