## Usage

For simple use and best accessibility support we recommend `NativeSelect`. 

<ul>
  <li>`NativeSelect` is recommend to use on products with limited space as it provides a better user experience.</li>
  <li>`NativeSelect` can be used to filter or sort content on a page.</li>
  <li>`NativeSelect` can be used to submit data.</li>
</ul>

```tsx
import { NativeSelect } from '@equinor/eds-core-react'

<NativeSelect label="Label text" id="native-select">
    <option>First option</option>
    <option>Second option</option>
    <option>Third option</option>
</NativeSelect>
```
