## Usage

**Selection controls**

<ul>
    <li>Allow users to select options, make decisions and set preferences.</li>
    <li>Are to be visible and understandable at a quick glance.</li>
    <li>Are always accompanied with a clear label.</li>
</ul>

**Radio**

<ul>
    <li>Exposes all available options.</li>
    <li>Makes it easy to compare available options.</li>
</ul>

```tsx
import { Radio } from '@equinor/eds-core-react'

<Radio label="Label text" />
```
