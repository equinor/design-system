## Installation 

Install the packages in your project directory with your package manager (npm/yarn/pnpm).
```shell
npm install @equinor/eds-core-react
```

## Usage

```tsx
import { Button } from '@equinor/eds-core-react'
<Button>Click me!</Button>
```
## Basic

The `Button` come with four variants; `contained`, `outlined`, `ghost` and `ghost_icon`.
<Story id="inputs-button--basic" />

## Icon button

The `ghost_icon` variant is meant for standalone icon buttons using the `<Icon>` component.


<Story id="inputs-button--ghost-icon" />


```tsx
<Button variant="ghost_icon">
  <Icon name="save" title="save action"></Icon>
</Button>
```
