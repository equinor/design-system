## Usage

The `Button.Toggle`
<ul>
  <li>emphasizes groups of related <code>Toggle</code> buttons,</li>
  <li>controls the selected state of its child buttons when given its own <code>selectedIndexes</code> prop.</li>
</ul>

```tsx
import { Button } from '@equinor/eds-core-react'
import { save } from '@equinor/eds-icons'

<Button.Toggle aria-label="actions">
  <Button aria-label="save action">
    <Icon data={save} title="Ghost icon save"></Icon>
  </Button>
  <Button aria-label="save action">
    <Icon data={save} title="Ghost icon save"></Icon>
  </Button>
  <Button aria-label="save action">
    <Icon data={save} title="Ghost icon save"></Icon>
  </Button>
</Button.Toggle>
```

## Accessibility

### ARIA

<ul>
  <li>Button.Toggle sets <code>aria-pressed="bool"</code> according to the button state.</li>
  <li>You should provide an accessible label with <code>aria-label="label"</code>, <code>aria-labelledby="id"</code> or <code>&lt;label&gt;</code>.</li>
</ul>
