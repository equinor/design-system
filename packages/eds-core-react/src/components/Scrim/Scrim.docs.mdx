# Scrim

Is a temporary visual effect that fades the general interface while allowing the user to focus on an overlay.

## Usage

A `Scrim` shows behind all other surfaces in an app, displaying contextual and actionable content.

<ul>
    <li>The scrimâ€™s back layer is persistent, displaying controls and content that relate to the front layer.</li>
    <li>The scrim focuses attention on one layer at a time.</li>
    <li>The scrim is displayed at full width and holds primary content.</li>
    <li>The scrim can be closed by either interacting with the front layer, using `esc` button or tapping a conceal affordance on either layer.</li>
</ul>

```tsx
import { Scrim } from '@equinor/eds-core-react'

const Demo = () => {
    const [isOpen, setIsOpen] = useState(false)
      const handleOpen = () => {
        setIsOpen(true)
      }
      const handleClose = () => {
        setIsOpen(false)
      }

    return (
        <>
      <Button onClick={handleOpen}>Open</Button>
      <Scrim onClose={handleClose}>
        <Button onClick={handleClose}>Close</Button>
      </Scrim>
    </>
  )
}
```
