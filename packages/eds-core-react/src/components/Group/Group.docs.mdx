## Usage

The buttons can be grouped by wrapping them with the `ButtonGroup` component.

```tsx
import { Button, Group } from '@equinor/eds-core-react'

<Group>
  <Button>Click me!</Button>
  <Button>Click me!</Button>
  <Button>Click me!</Button>
</Group>
```
