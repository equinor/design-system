## Usage

The buttons can be grouped by wrapping them with the `ButtonGroup` component.

```tsx
import { Button, Group } from '@equinor/eds-core-react'

<Group>
  <Button>Click me!</Button>
  <Button>Click me!</Button>
  <Button>Click me!</Button>
</Group>
```

### Accessibility

In addition, `Group` should be given an explicit label, as most assistive technologies will otherwise not announce them, 
despite the presence of the correct role attribute. 
In the examples provided here, we use `aria-label`, but alternatives such as `aria-labelledby` can also be used.
