## Usage

The buttons can be grouped by wrapping them with the `Group` component.

```tsx
import { Button, Group } from '@equinor/eds-core-react'

<Group aria-label="button group">
  <Button>Click me!</Button>
  <Button>Click me!</Button>
  <Button>Click me!</Button>
</Group>
```

### Accessibility

#### ARIA
<ul>
  <li>Group has <code>role="group"</code>. </li>
  <li>You should provide an accessible label with <code>aria-label="label"</code>, <code>aria-labelledby="id"</code> or <code>&lt;label&gt;</code>.</li>
</ul>

#### Keyboard

The button behavior follows standard keyboard semantics.
