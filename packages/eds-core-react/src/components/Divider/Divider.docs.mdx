# Divider
Horizontal, thin line.

## Installation 

Install the packages in your project directory with your package manager (npm/yarn/pnpm).
```shell
npm install @equinor/eds-core-react
```

## Usage

The `Divider` component separates content into clear groups. <br />
 <ul>
    <li>Divider should be noticeable in a layout, but not clashing.</li>
    <li>Dividers should only be used if elements cannot be split using white space.</li>
    <li>Use dividers occasionally, to create groupings rather than separate items.</li>
  </ul>
