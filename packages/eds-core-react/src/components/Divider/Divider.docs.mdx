## Usage

The `Divider` brings clarity to layout by grouping and dividing content into groups. <br />
 <ul>
    <li>Dividers should be noticeable in a layout, but not clashing.</li>
    <li>Dividers should only be used if elements can't be split using white space.</li>
    <li>Use dividers occasionally, to create groupings rather than separate individual items.</li>
    <li>Dividers can be used to establish rhythm and hierarchy.</li>
  </ul>

  ```tsx
import { Divider } from '@equinor/eds-core-react'

<Divider />
```

## Examples

### Full-bleed dividers

Full-bleed dividers separate content into segments and span the entire length of a layout.
<ul>
  <li>Separates list and layout elements.</li>
  <li>Indicates where an element may expand.</li>
  <li>Helps separate individual tiles.</li>
</ul>

### Inset dividers

Use inset dividers to separate related content. 
Use inset dividers when there are anchoring elements such as an icon or avatar.
