# Divider
Horizontal, thin line.

## Installation 

Install the packages in your project directory with your package manager (npm/yarn/pnpm).
```shell
npm install @equinor/eds-core-react
```

## Usage

The `Divider` brings clarity to layout by grouping and dividing content into groups. <br />
 <ul>
    <li>Dividers should be noticeable in a layout, but not clashing.</li>
    <li>Dividers should only be used if elements can't be split using white space.</li>
    <li>Use dividers occasionally, to create groupings rather than separate individual items.</li>
    <li>Dividers can be used to establish rhythm and hierarchy.</li>
  </ul>
