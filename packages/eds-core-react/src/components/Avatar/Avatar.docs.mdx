# Avatar

Avatars use images to visually represent people or companies.<br />
The avatar does not exist in Figma as a separate component.

## Usage

<ul>
    <li>Place avatars alongside additional information about the people or companies the avatars represent.</li>
    <li>Avatars can be displayed in lists that are searchable and selectable.</li>
    <li>Use inline avatars to mention people in a sentence.</li>
</ul>

```tsx
import { Avatar } from '@equinor/eds-core-react'

<Avatar />
```
