## Usage


Use the `EdsProvider` to encapsulate your whole application.
```typescript
import { EdsProvider } from '@equinor/eds-core-react'

<EdsProvider>
// Your app here
</EdsProvider>
```


## Accessibility

Be aware that only setting `density` to `compact` and not providing option to switch, breaks comformity to WCAG 2.1 guidelines and makes it harder to use the application for users with either permanent disabilities, or temporary disabilities.


## Examples

Examples of how to change density in your application using EDS components.

#### Nested density switcher

Use `useEds` to get & set `EdsProvider` values.  

⚠️ `useEds` only works from nested component inside `EdsProvider` as its built using [React Context](https://reactjs.org/docs/context.html)
