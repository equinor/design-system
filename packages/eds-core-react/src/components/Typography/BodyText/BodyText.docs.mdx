# BodyText
This component is currently available in the beta release of eds-core-React: <br />
`npm install @equinor/eds-core-react@beta` <br />
It requires the new eds 2.0 design tokens to be included in your app as css variables:
```tsx
//App.tsx or similar
import '@equinor/eds-tokens/css/variables.css'
```
`BodyText` uses the `Inter` font. We will likely bundle this on the equinor EDS CDN before the full release, but for now it is available from google fonts:
```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Inter:wght@100..900&display=swap"
  rel="stylesheet"
/>
```

## Usage
Text for use on general surfaces such as body, cards, popovers, sidesheets etc.

```tsx
import { BodyText } from '@equinor/eds-core-react'
 <BodyText size="lg">This is text</BodyText>
```
