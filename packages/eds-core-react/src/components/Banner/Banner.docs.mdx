## Usage

A banner displays an important, brief message, and provides actions for users to address. 
It requires a user action to be dismissed.

<ul>
    <li>Banners are interruptive, but their level of interruption should match 
        the information they hold and the context in which they appear.</li>
    <li>Only one banner should be shown at a time.</li>
</ul>

```tsx
import { Banner } from '@equinor/eds-core-react'

<Banner>
  <Banner.Message>
    Banner message comes here.
  </Banner.Message>
</Banner>
```

### Text and action

<ul>
  <li>Buttons in banners should directly relate to a banner’s message and clearly represent the banner’s action.</li>
  <li>Buttons must be labelled with text, not icons, for clarity.</li>
  <li>Place buttons under the banner message, or on the same line if there is enough room to fit both.</li>
</ul>
