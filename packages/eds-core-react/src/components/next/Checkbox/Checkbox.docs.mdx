## Usage

```tsx
import { Checkbox } from '@equinor/eds-core-react/next'

// Basic checkbox
<Checkbox label="Accept terms" />

// Checkbox group
<fieldset>
  <legend>Select options</legend>
  <Checkbox label="Option 1" name="group" value="1" />
  <Checkbox label="Option 2" name="group" value="2" />
</fieldset>
```

## Accessibility

The Checkbox component requires either a visible `label` prop or an `aria-label` to provide an accessible name. Use the `indeterminate` prop to represent a partially-checked state (e.g., "select all" when only some items are selected).
