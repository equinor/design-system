# Input

Text input and textarea component for forms and user input.

> **Note:** This is a [Next EDS 2.0 component](/docs/next-eds-2-0-introduction--docs). See the introduction guide for information about installation, usage, and the EDS 2.0 token system.

Intended to be used if you need more flexibility than the wrapped TextField or select components provide.

## Usage

```tsx
import { Input } from '@equinor/eds-core-react/next'

;<Input placeholder="Enter text" />
```
