## Usage

```tsx
import { Radio } from '@equinor/eds-core-react/next'

<fieldset>
  <legend>Select option</legend>
  <Radio label="Option 1" name="group" value="1" />
  <Radio label="Option 2" name="group" value="2" />
</fieldset>
```

## Accessibility

The Radio component requires either a visible `label` prop or an `aria-label` to provide an accessible name. Always group related radios under a `<fieldset>` with a `<legend>` describing the choice.
