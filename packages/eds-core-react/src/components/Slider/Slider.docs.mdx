## Usage

Sliders reflect the current state of the settings they control.

<ul>
    <li>Sliders should be used for making selections from a range of values.</li>
    <li>When interacting with a slider, changes should be reflected back to a user immediately.</li>
    <li>Sliders should present the full range of choices that are available to a user.</li>
</ul>

```tsx
import { Slider } from '@equinor/eds-core-react'

<Slider value={4} min={0} max={10} ariaLabelledby="simple-slider" />
```
