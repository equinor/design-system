## Usage

The buttons can be grouped by wrapping them with the `ButtonGroup` component.

```tsx
import { Button, ButtonGroup } from '@equinor/eds-core-react'

<ButtonGroup aria-label="button group">
  <Button>Click me!</Button>
  <Button>Click me!</Button>
  <Button>Click me!</Button>
</ButtonGroup>
```

### Accessibility

#### ARIA
<ul>
  <li>ButtonGroup has <code>role="group"</code>.</li>
  <li>You should provide an accessible label with <code>aria-label="label"</code>, <code>aria-labelledby="id"</code> or <code>&lt;label&gt;</code>.</li>
</ul>

#### Keyboard
<ul>
  <li>Navigate between buttons with the tab key.</li>
  <li>The button behavior follows standard keyboard semantics.</li>
</ul>
