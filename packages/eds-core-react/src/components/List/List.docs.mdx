## Usage

Lists are a continuous group of text.

<ul>
    <li>Lists should be sorted in logical ways that make content easy to scan.</li>
    <li>Lists present content in a way that makes it easy to identify.</li>
</ul>

```tsx
import { List } from '@equinor/eds-core-react'

<List>
  <List.Item>List item</List.Item>
  <List.Item>List item</List.Item>
  <List.Item>List item</List.Item>
</List>
```
