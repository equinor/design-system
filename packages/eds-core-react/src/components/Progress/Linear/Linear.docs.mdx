## Usage

`Linear progress` indicators are great to focus attention to an area.

<ul>
  <li>Progress indicators inform users about the current loading state giving them insight into the process.</li>
  <li>Progress indicators look and animate in ways that reflect the status of a process. They are never simply decorative.</li>
  <li>Progress indicators should be applied to all cases of a process in a consistent format.</li>
  <li>Linear progress indicators are great for places like empty pages, at the bottom of the top bar, and in cards and lists.</li>
</ul>

```tsx
import { LinearProgress } from '@equinor/eds-core-react'

<LinearProgress />
```
