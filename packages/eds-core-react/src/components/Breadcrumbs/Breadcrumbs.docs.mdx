## Usage

`Breadcrumbs` are often placed horizontally before a page's main content.

  ```tsx
import { Breadcrumbs } from '@equinor/eds-core-react'

<Breadcrumbs>
    <Breadcrumbs.Breadcrumb href="#">
      Item1
    </Breadcrumbs.Breadcrumb>
    <Breadcrumbs.Breadcrumb href="#">
      Item2
    </Breadcrumbs.Breadcrumb>
    <Breadcrumbs.Breadcrumb href="#" aria-current="page">
      Item3
    </Breadcrumbs.Breadcrumb>
</Breadcrumbs>
```

### Third-party routing libraries

Use the `as` prop on `Breadcrumbs.Breadcrumb` to utilize third party routers such as react-router with `Breadcrumbs`

```tsx
import { Link } from "react-router-dom"
import { Breadcrumbs } from '@equinor/eds-core-react'

<Breadcrumbs>
  <Breadcrumbs.Breadcrumb as={Link} to="/">
    Home
  </Breadcrumbs.Breadcrumb>
  <Breadcrumbs.Breadcrumb as={Link} to="/about" aria-current="page">
    About
  </Breadcrumbs.Breadcrumb>
</Breadcrumbs>
```
