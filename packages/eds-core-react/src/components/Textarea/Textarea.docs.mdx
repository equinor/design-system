## Usage

`Textarea` is a multi-line text field, useful for larger amounts of text.

<ul>
    <li>In its simplest form is a multi-line text-field, useful for larger amounts of text.</li>
    <li>**Label text** - every <code>Textarea</code> should have a label.</li>
</ul>

```tsx
import { Textarea } from '@equinor/eds-core-react'

<Textarea />
```

## Examples

<ul>
  <li>
    **Label text** informs users as to what information is requested for the textarea.
    Every textarea should have a label.
  </li>
  <li>
    **Helper text** gives additional guidance about the input field, such as how
    it will be used. It should only take up a single line.
  </li>
</ul>

### Multiline with fixed height

Note that a multiline with fixed `height` overwrites the `rows` property. `rowsMax` overwrites fixed `height`,
so it is not recommended to use those together.
