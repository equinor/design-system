## Usage

`TextField` should be discoverable, clear and efficient.

<ul>
  <li>Text fields should stand out and point that users can input information.</li>
  <li>Text field states should be <strong>clearly</strong> distinguish from one another.</li>
  <li>Text fields should make it easy to understand the required information and to address any errors.</li>
</ul>

  ```tsx
import { TextField } from '@equinor/eds-core-react'

<TextField />
```
