# Changelog - EDS 3.0 Beta Tokens

All notable changes to the EDS 3.0 beta tokens (`@equinor/eds-tokens@beta`) will be documented in this file.

These are the redefined tokens produced by the Tokens Studio pipeline, published under the npm `beta` dist-tag as a pinned `3.0.0-beta.N` series while stable 2.x remains on `latest`.
